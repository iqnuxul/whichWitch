## 🛠 技术栈

### 区块链
- **ZetaChain**: 主链和跨链功能
- **Solidity**: 智能合约开发
- **Hardhat**: 开发和测试框架
- **Ethers.js**: 区块链交互

### 前端
- **Next.js 14**: React 全栈框架
- **TypeScript**: 类型安全
- **Tailwind CSS**: 样式框架
- **Wagmi + RainbowKit**: Web3 连接

### 后端和AI
- **Node.js + Express**: API 服务器
- **通义千问 API**: AI 服务
- **IPFS (Pinata)**: 去中心化存储
- **TensorFlow.js**: 客户端AI处理

### 数据存储
- **Supabase**: 用户数据和缓存
- **IPFS**: 内容存储
- **区块链**: 核心数据和状态
